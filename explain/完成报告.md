## 项目架构理解

### 核心组件
1. **Gateway** - WebSocket 控制平面
   - 网关调用和认证
   - Webhook 处理和映射
   - 客户端连接管理

2. **Channels** - 消息通道
   - 通道注册表和元数据
   - 配置匹配和解析
   - 路由和门控

3. **Agents** - AI 代理执行
   - 缓存追踪
   - 工具调用
   - 会话管理

4. **Routing** - 消息路由
   - 会话键构建和解析
   - 会话隔离
   - 身份链接

5. **Infra** - 基础设施
   - 设备身份
   - 主目录解析
   - 状态管理

6. **Providers** - 模型提供商
   - OAuth 登录
   - 令牌管理

### 数据流
```
外部请求 → Gateway 认证 → 配置匹配 → 路由解析 → Agent 执行
  ↓                                          ↓
通道 ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← Provider API
  ↑
会话键解析
```
